# Hydrogen Atom Probability Density Visualization

## 📌 Overview
This project visualizes the probability density of an electron in a hydrogen atom using quantum mechanics.

## ⚛️ Features
- Computes radial and angular wavefunctions
- Uses spherical harmonics
- Generates 2D orbital density plots

## 📁 Project Structure
- `src/` → Python code
- `figures/` → Output plots
- `report/` → LaTeX PDF report

## ▶️ How to Run

```bash
pip install numpy matplotlib scipy
python src/hydrogen_orbital.py
